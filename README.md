# 津山高専 HR管理システム

このプロジェクトは、津山高専の生徒と教員向けに作成されたHR管理システムです。生徒は日々の学習や活動を記録することができ、教員は生徒の記録を確認・管理することができます。

## 機能概要

### 生徒向け機能
- ログイン機能
- 学習日誌の作成
- 自分の日誌の確認

### 教員向け機能
- 全生徒の日誌の閲覧
- 提出された日誌の統計情報の表示

---

## 必要な環境

- Python 3.7以上
- Flask
- openpyxl（オプション、Excelファイルの読み書きに必要）

## セットアップ手順

1. **リポジトリのクローン**
   ```bash
   git clone <リポジトリのURL>
   cd project
   ```

2. **仮想環境の作成と有効化**
   ```bash
   python -m venv venv
   # Windowsの場合
   venv\Scripts\activate
   # macOS/Linuxの場合
   source venv/bin/activate
   ```

3. **依存関係のインストール**
   ```bash
   pip install -r requirements.txt
   ```

4. **必要なディレクトリとファイルの作成**
   - `data`ディレクトリを作成し、`users.xlsx`と`reports.xlsx`を配置してください。
   - `users.xlsx`のフォーマット：
     | ユーザー名 | パスワード | 役割（student/teacher） |
     |------------|------------|-------------------------|
     | user1      | password1  | student                |
     | teacher1   | password2  | teacher                |
   - `reports.xlsx`のフォーマット：
     | ユーザー名 | 日付       | タイトル       | 内容               | 投稿日時               |
     |------------|------------|----------------|--------------------|------------------------|
     | user1      | 2026-01-01 | 学習内容       | 今日は数学を勉強しました | 2026年01月01日 10:00:00 |

5. **アプリケーションの起動**
   ```bash
   python hello_server.py
   ```

6. **ブラウザでアクセス**
   デフォルトでは、以下のURLでアプリケーションにアクセスできます：
   ```
   http://127.0.0.1:5000/
   ```

## 使用方法

### ユーザーの登録
1. `data/users.xlsx`ファイルに、ユーザー名、パスワード、役割（`student`または`teacher`）を記載してください。
2. `student`は生徒用、`teacher`は教員用のアカウントになります。

### ログイン
1. `/` にアクセスしてログイン画面を開きます。
2. ユーザー名とパスワードを入力してログインします。

### 生徒モード
- ログイン後、日誌を作成するためのフォームが表示されます。
- 過去に作成した日誌を一覧で確認できます。

### 教員モード
- ログイン後、全生徒が提出した日誌の一覧や統計情報を確認できます。

## 注意事項

1. **データの保存先**  
   - 日誌データは`data/reports.xlsx`に保存されます。
   - ユーザー情報は`data/users.xlsx`に保存されます。
   - 必要に応じて、これらのファイルをバックアップしてください。

2. **セキュリティ**  
   - 本システムはデモ用のため、`app.secret_key`は必ず変更してください。
   - 実運用する場合は、HTTPSの導入やパスワードのハッシュ化を行ってください。

3. **デバッグモードについて**  
   - 現在の設定では`debug=True`になっています。本番環境では`debug=False`に変更してください。
